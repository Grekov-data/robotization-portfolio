# Лендинг-портфолио: Роботизация бизнес-процессов вокруг 1С

Одностраничный сайт-визитка. Всё в одном файле `index.html` (стили встроены, зависимостей нет) — легко открыть, отредактировать и бесплатно разместить.

## Как посмотреть

Просто открой `index.html` двойным кликом — он откроется в браузере.

## Что нужно заменить перед публикацией

Открой `index.html` и найди эти места (можно через поиск Ctrl+F):

1. **Контакты** (внизу страницы, блок `id="contact"`):
   - `your@email.ru` → твоя почта
   - `https://t.me/your_username` → твой Telegram
2. **Бренд/имя** в шапке — при желании поставь своё имя вместо «Роботизация 1С».
3. **Числа в кейсах** уже проставлены по твоим данным. Если какие-то поменяются — правь прямо в тексте кейса.

## Как выложить в интернет бесплатно (по возрастанию сложности)

1. **GitHub Pages** — залить репозиторий на GitHub, включить Pages в настройках. Получишь ссылку вида `имя.github.io/robotization-portfolio`.
2. **Netlify / Cloudflare Pages** — перетащить папку в веб-интерфейс, получить ссылку за минуту. Можно подключить свой домен.
3. **Свой домен** — купить короткий домен (например, на reg.ru) и привязать к любому из вариантов выше — выглядит солиднее в аутриче.

Скажи, каким способом хочешь публиковать — помогу пошагово.
