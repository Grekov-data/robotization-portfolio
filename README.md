# Лендинг-портфолио: Роботизация бизнес-процессов вокруг 1С

Одностраничный сайт-визитка. Всё в одном файле `index.html` (стили встроены, зависимостей нет) — легко открыть, отредактировать и бесплатно разместить.

## Как посмотреть

Просто открой `index.html` двойным кликом — он откроется в браузере.

## Структура

```
robotization-portfolio/
├─ index.html   # весь лендинг (разметка + стили)
└─ README.md    # этот файл
```
